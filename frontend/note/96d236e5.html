一、什么是 vdom

  • 用 JS 模拟 dom 结构
  • dom 变化的对比，放在 JS 层来做
  • 提高重绘性能

二、设计一个需求场景

遇到的问题

  • dom 操作是“昂贵”的，js 运行效率高
  • 尽量减少 dom 操作，而不是“推倒重来”
  • 项目越复杂，影响就越严重
  • vdom 即可解决这个问题

  • 这个“找出”的过程，就需要 diff 算法
