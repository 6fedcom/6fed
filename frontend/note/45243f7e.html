  • 如何中断事件的传播？

    • cancleBubble = true IE取消冒泡

  • 取消事件默认效果：

    • returnValue = false IE 取消事件效果
    • defaultPrevent() w3c取消事件效果

  • 冒泡写法停止传播 从下往上

    • returnValue = false //IE 取消事件效果
