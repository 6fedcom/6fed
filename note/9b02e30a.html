Sublime Text添加插入带当前时间作者详细说明

2016-06-03
  • 工欲善其事必先利其器

创建插件：

Tools → New Plugin:

保存为Sublime Text 2\Packages\User\ addInfo.py

创建快捷键：

Preference → Key Bindings - User:

此时使用快捷键 ctrl+shift+, 即可在当前光标处插入当前时间，如下：

最后更新时间：2018-10-23 16:47:45
这里可以写作者留言，标签和 hexo 中所有变量及辅助函数等均可调用，示例：https://ru23.com/note/9b02e30a.html
